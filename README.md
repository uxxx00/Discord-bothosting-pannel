🚀 Discord Bot Hosting Panel
A modern and powerful Discord Bot Hosting Panel that allows users to easily deploy, manage, and control their bots with a clean and user-friendly interface.
✨ Features
🔐 Authentication System
Login with Google and GitHub
Secure session handling
🧠 Admin Panel
Full control over users and bots
Manage credits, plans, and system settings
💰 Credits System
Users can earn or buy credits
Credits used for hosting resources
🎁 Promo Codes
Create and redeem promo codes
Flexible reward system
🤖 Bot Hosting
Upload and run Discord bots بسهولة
Real-time status monitoring
Logs and error tracking
⚡ Performance
Optimized for speed and stability
Scalable backend system
🛠️ Tech Stack
Frontend: React / Next.js
Backend: Node.js
Database: MongoDB / PostgreSQL
Authentication: OAuth (Google & GitHub)

🚀 Getting Started
Bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
npm install
npm run dev
⚙️ Configuration
Create a .env file and add:
Environment
GOOGLE_CLIENT_ID=your_google_id
GITHUB_CLIENT_ID=your_github_id
DATABASE_URL=your_database
SESSION_SECRET=your_secret
